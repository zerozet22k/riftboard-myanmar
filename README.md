# RiftBoard Myanmar

RiftBoard Myanmar is a Myanmar-focused League of Legends stats, rankings, and player tracking platform built with Next.js and TypeScript.

This project was created as a personal portfolio piece to showcase frontend development, UI structuring, and product-oriented web application design.

## Features

- Player profile pages
- Ranking and leaderboard views
- Match and performance tracking
- Player search
- Localized experience for Myanmar users

## Tech Stack

- Next.js
- TypeScript
- CSS / PostCSS
- ESLint
