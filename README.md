# RiftBoard Myanmar

A Myanmar-focused League of Legends stats, rankings, and player tracking platform built with Next.js and TypeScript.

RiftBoard Myanmar is designed to provide a cleaner, more accessible, and localized experience for players who want to explore rankings, track performance, search for players, and follow competitive progress in one place.

## Live Demo

[rft-board-myanmar.vercel.app](https://rift-board-myanmar.vercel.app)

## Overview

RiftBoard Myanmar is a web application centered around player visibility, leaderboard access, and performance tracking for the Myanmar League of Legends community.

The project aims to make competitive player data easier to browse and understand through a dedicated interface tailored for local users.

## Features

- Player profile pages
- Ranking and leaderboard views
- Match and performance tracking
- Summoner or player search
- Localized experience for Myanmar users
- Fast, modern web interface built with Next.js

## Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** CSS / PostCSS
- **Linting:** ESLint
